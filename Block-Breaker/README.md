# Block Breaker

## Instructions to Download and Open the Game
1. Clone the repository to your local machine using the command:
   ```bash
   git clone https://github.com/dillonparker8/Dillons-games.git
   ```
2. Navigate to the 'Block-Breaker' subfolder:
   ```bash
   cd Dillons-games/Block-Breaker
   ```
3. Open the game using your preferred IDE or editor that supports the programming language used (e.g., Python, Java).

## Game Rules
- The objective of the game is to break all the blocks on the screen.
- Players control a paddle by moving it left or right to bounce the ball towards the blocks.
- Each block, when hit by the ball, will disappear and award points.
- If the ball falls below the paddle, the player loses a life. Players have a limited number of lives.

## Gameplay Instructions
- Use the arrow keys or A/D keys to move the paddle left and right.
- Press the space bar to launch the ball at the start of the game.
- Keep the ball in play by bouncing it off the paddle and hitting the blocks until all blocks are destroyed! 

Good luck and have fun!
